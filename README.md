# HR Tool

Hệ thống Hỗ trợ Tuyển dụng & Quản lý Hồ sơ Ứng viên.
Đây là phiên bản MVP hỗ trợ tải lên CV, trích xuất thông tin, và tạo CV chuẩn hóa dưới dạng PDF/DOCX.

## Cấu trúc dự án
- `backend/`: FastAPI + PostgreSQL
- `frontend/`: React + Vite + TailwindCSS
- `docs/`: Tài liệu (PRD, BRD)
